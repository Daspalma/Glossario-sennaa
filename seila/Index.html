Glossário muito incrível e eficiente caso haja uma prova semana que vem e eu tenha que usar este INCRÍVEL glossário para uma ajudinha extra!

  • <html> Define o documento HTML.
  • <head> Contém metadados e links para estilos/scripts.
  • <body> Contém o conteúdo visível da página.
  • <h1> ... <h6> Cabeçalhos de diferentes níveis.
  • <p> Define um parágrafo.
  • <a> Cria links de navegação.
  • <img> Exibe imagens.
  • <div> Container genérico.
  • <span> Container para texto em linha.
  • <form> Cria formulários.
  • color Define a cor do texto.
  • background-color Define a cor de fundo.
  • font-size Define o tamanho da fonte.
  • font-family Define a família de fontes.
  • margin Espaço externo do elemento.
  • padding Espaço interno do elemento.
  • border Define a borda.
  • width Define a largura.
  • height Define a altura.
  • display Define o tipo de exibição (block, flex, inline...(alinhar imagem tambem)).
  • if serve para verificar uma condição e executar um bloco de código só se for verdadeira.
  • else serve para tratar o caso contrário, ou seja, executar outro bloco de código quando a condição do if for falsa.
