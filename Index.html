Glossario muito incrivel e eficiente caso haja uma prova semana que vem e eu tenha que usar este INCRIVEL glossario para uma ajudinha extra!

    Margin

    serve para criar um espaço externo ao redor de um elemento, separando-o de outros elementos na página.



    Div

    Contêiner genérico usado para agrupar outros elementos sem significado semântico.



    Footer

    Representa o rodapé de um documento ou seção, normalmente com informações como direitos autorais ou links.



    Class

    Atributo que permite aplicar estilos ou scripts a um ou mais elementos de uma página.



    Header

    Representa o cabeçalho de um documento ou seção, geralmente com título e navegação.



    Img

    Tag usada para exibir imagens em uma página web.



    Cursor

    Propriedade CSS que define o tipo de cursor (seta, mãozinha, etc.) ao passar o mouse sobre um elemento.



    Containers

    Elementos usados para agrupar conteúdo e controlar o layout (geralmente com div).



    Link

    serve para criar um espaço externo ao redor de um elemento, separando-o de outros elementos na página.
